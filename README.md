# hailstorm-w540

Configuration files of my W540 ThinkPad running NixOS.

## Notice
As always, replace all instances of 'puppy' with your actual username on your own machine.

## Specs
 - WM: niri
 - Bar: noctalia-shell
 - Shell: zsh
 - Editor: emacs
 - Distro: NixOS
 - Compositor: niri/wayland

## Nix Specification
This setup comes with **home-manager**, **stylix**, **nix-flatpak**, and **nix-index-database**, and much more, pre-installed and included in the flake.
